# SpaceSync — Resource Allocation System

A lightweight full-stack dashboard for Jahangirnagar University to manage booking of departmental resources (labs, libraries, equipment). Built for CSE 362 Lab Final.

> **Status:** Phase 1 scaffold. Full README (team roles, setup, API, Postman, screenshots) is filled in during Phase 7.

## Tech Stack
- **Backend:** Node.js, Express, Sequelize, MySQL
- **Frontend:** React + Vite, TailwindCSS, Axios, React Router, react-hot-toast

## Repository Layout
```
SpaceSync/
  backend/    # Express + Sequelize API
  frontend/   # React + Vite SPA
  docs/       # Schema screenshot & extras
  README.md
```
